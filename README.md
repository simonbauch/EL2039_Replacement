# EL2039_Replacement
A replacement PCB for the commonly failing EL2039 on the AC boards of 3458As. 

This is accomplished using a THS4631DGNR from TI as the opamp in the AC amp on the A2 PCB. (like newer 3458As do) To stay in the proper operating voltage range a 3V Zener dropper is used on the -17V rail.

This replacement board passes Selftest and ACAL AC, more tests haven't been done yet. It fits mechananically below the shield.
